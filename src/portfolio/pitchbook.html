Background

PitchBook provides customers with private market data through the PitchBook Platform, a suite of award-winning software applications and financial analysis tools.

Goals

  • Improve PitchBook’s data coverage and acquisition
  • Find companies and data points that competitors cannot
  • Increase quality of data
  • Use data science to provide insights on quality of data
  • All happens at a rate that exceeds what a human could perform through advanced computer science and machine learning engineering methods

Projects

The following are a collection of tools and applications I designed and developed to enhance data coverage, help acquire exclusive company insights, improve data quality using data science, and achieve rapid processing beyond human capabilities through advanced computer science and machine learning methods

Screenshot Generator

  • Built to improve private company survey process
  • Time needed to create 120 screenshots for survey emails:
    • Manually: ~90 minutes
    • Using Screenshot Generator: ~4 minutes
    • 2,250% speed improvement
  • Process is automatic; researcher can do other tasks during 4 minutes
  • Used temporary user interface
  • Integrated into internal workflow software, further improving speed

Primary Report Tool

  • Updated Report Tool interface for Primary Research team to have in front of them during calls
  • Updated user interface to increase legibility and usability
  • Displaying most relevant data points with customized & prioritized ordering and highlighting
  • Implemented “Export to Excel” feature to streamline note-taking and data entry accuracy

Survey Profile Editor

  • Redesigned and enhanced survey tool for better user interaction with data
  • Concept gives many more people a preview of what PitchBook looks like
  • Clean, intuitive interface allows for simple data editing and confirmation
  • Introductory guided interaction flow will be added to ensure most important data points are entered/confirmed by survey contact

Profile Editor is a replica of PitchBook platform, but with editable fields (hovering over value shows editability)

Clicking on a value will transform it into an appropriate input field (textbox, dropdown menu, etc.)

News as a Service (NaaS)

The NaaS interface was used as a primary application by full-time researchers and research managers, in order to identify key sentences about company events and key financial data points (e.g. revenue, EBITDA, funding rounds, employee count) from various news sources, capture that data, and validate/improve the accuracy of the machine learning models.
